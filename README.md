# CKA 2026 Real-World Labs (Killercoda)

This repo is a *Killercoda course* containing 17 exam-style CKA practice scenarios.

## How to use on Killercoda
1. Create a GitHub repo and push this folder as-is.
2. In Killercoda: **Creator → Repository** and add your repo.
3. Copy the **Deploy Key** shown by Killercoda and add it to your GitHub repo: **Settings → Deploy keys** (read-only).
4. Every push to the configured branch will update the scenarios on Killercoda.

## Local quick test (optional)
You can open any scenario folder and review:
- `index.json` (scenario wiring)
- `intro.md` (prompt)
- `scripts/background.sh` (setup/reset)
- `scripts/verify.sh` (pass/fail checks)

> Notes:
> - Some scenarios include TODOs in verification/setup where you may want to tune assumptions (node names, installed add-ons).
